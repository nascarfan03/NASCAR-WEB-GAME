# NASCAR Web Game
never using github copilot again 😆## Description
Experience the thrill of NASCAR racing right in your browser! The NASCAR Web Game is an interactive racing game that allows players to compete against each other or against the clock.

## Features
- Realistic car physics
- Multiple tracks to choose from
- Customizable car skins
- Compete with friends or AI opponents

## Controls
- **Arrow Keys**: Steer your car
- **Spacebar**: Accelerate
- **R**: Reset the current race
- **C**: Customize your car

## Loading Your Custom Car
To load your custom car, place the car image in the `assets/cars/` directory and use the customization menu in-game.
